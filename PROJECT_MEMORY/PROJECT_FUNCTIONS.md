# PROJECT_FUNCTIONS.md

## Компоненты

- `Header` — навигация и телефон.
- `Hero` — главный экран с названием Абьюз Бьюти, миссией, рейтингом, количеством оценок и графиком.
- `Advantages` — преимущества студии без привязки к удалённым услугам.
- `Services` — карточки только актуальных услуг.
- `PriceList` — цены только из утверждённого списка.
- `Masters` — направления мастеров без выдуманных персональных данных.
- `Gallery` — интерьер и работы с волосами.
- `Reviews` — рейтинг 5.0 и 65 ratings без выдуманных отзывов.
- `Promotions` — только акция по стрижке и окрашиванию.
- `FAQ` — вопросы только по актуальным услугам, графику, оплате и Wi-Fi.
- `BookingForm` — форма записи с актуальным списком услуг и временем 10:00–19:30.
- `Contacts` — 2ГИС, WhatsApp, Telegram, MAX, оплата, Wi-Fi.
- `FloatingButtons` — звонок, WhatsApp, MAX и запись.
- `Footer` — актуальные контакты и услуги.

## Важные функции

- `scrollTo` — плавный переход к секциям.
- `validate` в `BookingForm` — проверка формы.
- `handleSubmit` в `BookingForm` — подготовка заявки, EmailJS оставлен без публичного получателя.
