# PROJECT_RULES.md

## Правила проекта

- Перед изменениями проверять `src/data`, `src/components`, `index.html`, `PROJECT_MEMORY` и публичные README.
- Не добавлять услуги, цены, мастеров, акции, контакты или тексты, которых нет в актуальных данных.
- Все контакты менять только в `src/data/config.ts`.
- Все услуги менять только в `src/data/services.ts`.
- Все цены менять только в `src/data/prices.ts`.
- Если данные отсутствуют, не заменять их выдуманными.
- Не использовать направления вне актуального списка услуг.
- Не использовать фразы о работе вне утверждённого графика.
- График везде только: Ежедневно 10:00–20:00, По предварительной записи.
- После каждого изменения обновлять файлы PROJECT_MEMORY вручную в рамках того же изменения.

## Архитектура

- `src/App.tsx` — порядок секций.
- `src/components` — UI секции.
- `src/data` — единственный источник контента.
- `public/images` — заменяемые изображения.
